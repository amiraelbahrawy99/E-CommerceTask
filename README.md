# E-CommerceTask

Testing functionality of Cart Checkout for authorized users


### Prerequisites

* JDK 1.8
* IDE (IntelliJ OR Eclipse)

### Running the tests

* ** testng.xml ** can be found in the root path of the project
* ** Command Line ** mvn clean verify
